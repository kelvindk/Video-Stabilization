matrix_traits<Matrix>::scalar_type

typedef /*user-defined*/ scalar_type;

This is the member type that user-defined (partial) specializations of the matrix_traits template define to specify the scalar type of a matrix type. Valid scalar types are value types that satisfy the scalar requirements.
