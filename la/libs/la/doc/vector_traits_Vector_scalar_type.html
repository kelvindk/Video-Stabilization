vector_traits<Vector>::scalar_type

typedef /*user-defined*/ scalar_type;

This is the member type that user-defined (partial) specializations of the vector_traits template define to specify the scalar type of a vector type. Valid scalar types are value types that satisfy the scalar requirements.
