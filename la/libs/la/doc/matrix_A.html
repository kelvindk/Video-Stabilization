An expression of the form of m|A<R,C> can be used to access the element at row R and column C of a matrix object m. For example, the expression:

m|A<4,2> *= 42;

can be used to multiply the element at row 4 and column 2 of a matrix m by 2.

For convenience, there are also non-template overloads for R from 0 to 9 and C from 0 to 9; an alternative way to write the above expression is:

m|A42 *= 42;
