Boost LA

rotz_matrix

#include <boost/la/rotz_matrix.hpp>

namespace
boost
    {
    namespace
    la
        {
        template <int Dim,class T>
        -unspecified-return-type-
        rotz_matrix( T angle );
        }
    }

Requirements:

Dim>=2

Returns:

A matrix with Dim rows and Dim columns and scalar_type T which performs a rotation around the Z axis at angle radians.

See also: boost/la/rotz_matrix.hpp
